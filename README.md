

## Project setup -

### Download the code

### Set the .env File 
```bash
database credentials
email credentials
```

### Migrate Database Files
```bash
php artisan migrate
```

### Run Seeder Files
```bash
php artisan db:seed
```

### Run project
```bash
php artisan serve
```

### Run project in browser
```bash
http://127.0.0.1:8000
```
